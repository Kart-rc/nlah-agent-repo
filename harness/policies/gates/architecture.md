# Architecture Gate Checklist

Required for: new services, API design, data model changes, cross-service
integration, new infrastructure patterns, significant dependency introduction.
Verify the artifact provides:

- [ ] Options considered (more than one, treated fairly)
- [ ] Recommended approach with rationale
- [ ] Tradeoffs stated explicitly
- [ ] Failure modes identified
- [ ] Operational impact addressed (who runs it, how it is observed)
